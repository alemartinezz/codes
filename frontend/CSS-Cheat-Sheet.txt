Cascading Style Sheets at the most basic level it indicates that the order of CSS rules matter. 

.class
#id
*
element
element, element2 /*se copian las reglas a los 2*/
element element2 /*solo aplica a los element2 que estan ADENTRO de element*/
element > element2 /*solo aplica a los element2 que estan dentro de element en primera fila*/
element + element2 /* los element2 que estan DESPUES de element*/
:hover
:last-child
:first-child
!important (not recommended)


What seletors win out in the cascade depends on:
-Specificity
-Importance
-Source Order
